## Setup

1. Clone this Repository(make sure to have git installed)
   ```cmd
   git clone https://github.com/cjdevx-ai/rag-chatbot.git
   ```
2. Create the virtual environment:
   ```bash
   python -m venv .venv
   ```
3. Activate the virtual environment (using PowerShell):
